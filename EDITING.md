# Editing The Website

Open the editor from the GitHub repo folder:

`/Users/jd/Documents/GitHub/Portfolio/admin.html`

Recommended workflow:

1. Open `admin.html` in the browser.
2. Click `Open content.js` and choose `/Users/jd/Documents/GitHub/Portfolio/content.js`.
3. Edit content, add items, delete items, reorder rows, or hide/show sections.
4. Click `Save content.js`.
5. Open `/Users/jd/Documents/GitHub/Portfolio/index.html` to preview.
6. In GitHub Desktop, commit and push the changes.

If direct saving is unavailable in the browser, click `Download content.js` and replace the repo’s existing `content.js` file with the downloaded file.

Most future content changes should happen in `content.js` through the editor. The design lives in `styles.css`, and the rendering logic lives in `script.js`.
